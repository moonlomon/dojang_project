## 반복문 제어 : break, continue
### 반복문에서 break를 쓰면 루프에서 벗어날 수 있다.
```python
# while문 무한루프 빠져나오기
i = 0
while True:
    print('break 예시')
    i+=1
    if i==5:
        break #i가 5가 되면 루프 종료

# for문 5번째 루프에서 나오기
for i in range(1000):
    print(i+1,"번째 루프")
    if i==4:
        break #1000번 되기 전 5번째에서 루프 종료
```
