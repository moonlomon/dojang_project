## 딕셔너리 만들기
```python
# {키:데이터, 키:데이터} 이렇게 데이터를 묶으면 딕셔너리가 된다
person = {'이름' : 'john', '나이' : 25, '성별' : '남자'}

# 키에 리스트, 딕셔너리를 사용할 수 없다
a = {'a':1, 2:['x',2], True:2.3}
print(a) #출력: 'a': 1, 2: ['x', 2], True: 2.3}
b = {[1,2]:'a'}
print(b) #에러(TypeError): unhashable type: 'list'ㅋ

# 키가 중복되면 가장 뒤에(최근)값만 딕셔너리에 저장된다
_dic = {1:1, 1:2, 1:3}
print(_dic) #출력: {1: 3}
```

## dict()함수로 딕셔너리 만드는 4가지
```python
방법1 = dict({1:1, 2:2, 3:3})
방법2 = dict(a=1, b=2, c=3) # 주의: 키는 무조건 string으로 따옴표 사용x
방법3 = dict([('x':1), ('b':2), ('c':3)])
방법4 = dict(zip([1,2,3], ['a','b','c'])) # 주의: 리스트 대신 튜플 써도 됨
print(방법1, 방법2, 방법3, 방법4, sep:'\n') #출력:  {1: 1, 2: 2, 3: 3}
                                            #     {'a': 1, 'b': 2, 'c': 3}
                                            #     {'x': 1, 'b': 2, 'c': 3}
                                            #     {1: 'a', 2: 'b', 3: 'c'}
```

## 딕셔너리 활용하기
```python
# 딕셔너리 접근하기
a = {1:11, 2:22, 3:33}
a[1] = 111
print(a[1]) #출력: 111

# 키 혹은 값이 있는지 확인하기
_dic = {'a':1, 'b':2, 'c':3}
print('a' in _dic) #출력: True
print(1 in _dic.values()) #출력: True

# 딕셔너리 길이 구하기
a = {1:1, 2:2, 3:3}
print(len(a)) #출력: 3
```
