### 참이면 True 거짓이면 False
```python
# 숫자
10 == 10 #True
10 != 10 #False
10 <= 10 #True

# 문자
'python' == 'python' #True
'Python' == 'python' #False
```

### 값 비교 + 속성 비교 = is, is not
```python
# equal과 not equal은 값의 차이만 비교
print(1 == 1.0) #출력: True

print(1 == 1.0) #출력: False
```

### 논리 연산자: and, or, not
```python
# 
print(True and True) #출력: True
print(True and False) #출력: False
```
